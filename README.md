## 🚀 خطوات الحصول على APK (بدون أي برامج)

### 1️⃣ افتح github.com
- سجّل دخول أو اعمل account مجاني
- اضغط زر **+** فوق يمين → **New repository**
- اسم الـ repository: **okx-tracker**
- اختار **Public**
- اضغط **Create repository**

---

### 2️⃣ ارفع الملفات
- في الصفحة الجديدة اضغط **uploading an existing file**
- ارفع **كل الملفات والفولدرات** اللي في الـ ZIP ده
  - ⚠️ **مهم:** ارفع الملفات من **جوّا** الفولدر مش الفولدر نفسه
- اضغط **Commit changes**

---

### 3️⃣ شغّل GitHub Actions
- اضغط **Actions** من فوق
- هتلاقي **🏗️ Build OKX Tracker APK**
- لو مش اشتغل تلقائي: اضغط عليه → **Run workflow** → **Run workflow**
- استنى **3-5 دقائق** ✅

---

### 4️⃣ حمّل الـ APK
- لما يخلص وتشوف ✅ اخضر
- اضغط على الـ run
- في الأسفل هتلاقي **Artifacts**
- اضغط **OKX-Tracker-WearOS-APK**
- هينزل ملف ZIP — جوّاه `app-debug.apk`

---

### 5️⃣ ثبّت على الهاتف
1. روح **الإعدادات** → **الأمان** → فعّل **تثبيت من مصادر غير معروفة**
2. افتح ملف الـ APK وثبّته على الهاتف

---

### 6️⃣ ظهور على الساعة تلقائياً
- افتح **Galaxy Wearable** على الهاتف
- روح **App settings** → **Add apps**
- اختار **OKX Tracker** وثبّته على الساعة ✅
